The BeautifulSoup Class
=======================

.. currentmodule:: bs4

The :py:class:`bs4.BeautifulSoup` class is the main class that you work with. It is a child of the :py:class:`bs4.element.Tag` class. It is typically instantiated by passing in a string of HTML to the constructor.

.. code::

   response = requests.get(URL)
   soup = BeautifulSoup(response.text)
